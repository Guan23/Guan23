说明：
1、16个激光头的垂直角度是固定的，水平角的起始方向从缆线的对面开始算起，具体可查阅velodyne16的官方文档
2、Velodyne16有最小测程0.2m（貌似）
3、水平方向共分前后左右四个大方位，上下暂不考虑
前：0 < Y < 前边界
右：右边界 < X < 0
后：后边界 < Y < 0
左：0 < X < 左边界
其余没做限制的坐标均在最小测程内，即-0.2~+0.2之间
4、后续的滤波，以及分类还要看雷达的安装位置
5、data数据共5列，p[0]~p[4]分别为Y、X、Z(单位均为米)、反射强度(0~255)、激光头(0~15)
6、其中Y向前+后-，X向左+右-，Z向上+下-
7、假设立方体小车身宽40CM，长60CM，高20CM，激光雷达安装在车头中间部位
8、探测的方式有两种，一种是探测矩形范围，一种是探测扇形范围
探测矩形则通过约束前后左右边界来实现，探测扇形则通过约束斜率来实现
假设汽车左右对称，故此处只需约束一四象限的斜率即可（二三也可）
